边框粗细:
border-width: 0px;单位px

边框颜色:
border-color: red;

边框样式:
border-style: dashed;
solid为实线边框，dashed为虚线边框，dotted为点线边框，还有很多样式
三者复合形式，没有顺序。border：10px dashed red;
边框也可某一边单独设置，top为上，bottom为下，left为左，right为右

合并相邻的边框
border-collapse: collapse;

padding-right: 20px;
padding是内边距，是边框距离内容的距离，bottom left right
当设置盒子，每个小盒子文字字数不同时，可设置padding来使小盒子挤大，视觉效果更好，例如导航界面.
如果盒子本身没有设置width和height大小,则设置padding值不会撑开盒子大小
padding复合写法：
padding：5px ,padding后面跟一个值，上下左右都是5px
padding：5px 10px，padding后面跟两个值，上下5，左右10
padding：5px 10px 15px，padding后面跟3个值，上5，左右10，下15
padding：5px 10px 15px 20px，padding后面跟4个值，上5，右10，下15，左20，按照顺时针方向

margin: 20px;
外边距，与border-style一样有top、bottom等，根据实际控制。复合写法与padding一样
margin: 0 auto;
外边距可以让块级盒子水平居中显示，需满足的条件：
1.必须设置了盒子宽度width，
2.盒子左右外边距设置为auto
外边距合并，当上面的块元素设置下外边距为100px，下面的块元素设置上外边距为200px时，最后两个块元素上下距离为200px，他们的外边距会合并，取最大。解决方案为尽量只给一个块元素设置就行 */
对于两个是父子关系的块元素，即一个较大的里面有个较小的，则父亲会塌陷较大的外边距值，父元素设置20px，子元素设置10px，最后只有20px。解决方案：1、给父元素设置上边框，2、给父元素设置上内边框，3、为父元素添加overflow：hidden */

margin负值的巧妙运用：
1、给margin-left等添加边框大小一样的负值，可以消除几个盒子重叠时边框的变粗问题
2、当几个盒子浮动在同一行，当鼠标经过时添加其他颜色的边框 ：
如果盒子没有设定定位，则添加相对定位即可position：relative；如果盒子已有相对定位，则给：hover中加zidex:1即可
*/
圆角边框
border-radius:半径长度（决定边框有多弯）

例如在设计圆形时，先准备一个正方形，然后圆角边框数值设计为正方形边长的一半，可用px作为单位，或者直接写50% ，假如是圆角矩形，则半径长度设为高度的一半*/
可以设置4个数值，从左上角开始顺时针设置,两个数值则为对角线，例如：只想设置一个角则写成border-top-left-radius为左上角 */

注意点：
1.内边距和边框都会影响盒子大小，设置盒子大小时，需要考虑边框和页边距
2、写代码时，尽量不要给行内元素设置上下内外边距
3、 box-sizing：border-box，
在css中加上该语句，则盒子的大小只为width，不再累加上padding和border 其默认值为context-box，其盒子大小为三者相加
