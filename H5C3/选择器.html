选择器

1、标签选择器

标签名 { 属性 }

2、类选择器

.类名称{ 样式 }
类的命名有其规则和常用叫法，口诀：样式点定义，结构类调用,一个或多个，开发最常用。
一个标签可有多个类名，类名间用空格隔开class="red yellow”

3、id选择器

选择器:#id名 { }
调用时选择标签的id名相同的，id唯一，class不唯一

4、通配符选择器

*{ }
选中页面中的所有标签

5、后代选择器

父元素 子元素 ...(一层层往里写都可以，元素可以是标签、.类名、#id名) { 样式 }
选择父元素的所有子元素

6、子选择器

父元素>子元素{ }
选择父元素的第一个子元素

7、并集选择器

元素名，元素名 { }
该选择器的元素名喜欢竖着写,将同一级的标签改为相同的样式。例如：将div和h3同时修改：div，h3 { }

8、伪类选择器

a：名字{ }
a：link 选择未被访问的链接
a：visited 选择已被访问的链接
a：hover 选择鼠标位于其上的链接
a：active 选择鼠标按下未弹起的链接
LVHA的顺序不能颠倒，先写后写

9、:focus伪类选择器

主要针对于表单元素，把获得光标的表单元素选取出来
input:focus{ color:red; }

10、属性选择器

元素 [属性="属性值"], 或者: 元素 [属性]

11、结构伪类选择器

1、元素 ：first-child，选择选择父元素中第一个子元素
2、元素 ：last-child ，选择父元素中最后一个元素
3、元素 ：nth-child（数字或关键字或公式例如2n+1），选择父元素中的第n个元素，关键字even表示偶数，odd表示奇数，括号中写n表示选择了全部的子元素，n从0开始，n++
4、元素 ：first-of-type，选择第一个元素类型
5、元素 ：last-of-type，选择最后一个元素
6、元素 ：nth-of-type，选择第n个元素
前三个与后三个的区别是，前三个先看第几个，再去匹配相应元素，后三个是先看元素，再去找第一个元素，后三个多用于子元素的标签有不一样的情况

12、伪元素选择器

1、元素名：：before{}，在元素内部的前面插入内容
2、元素名：：after{}，在元素内部的后面插入内容
before和after创建一个元素，但是属于行内元素
必须有content属性，可以为空，也可以写里面的内容
权重和标签选择器一样为1 ，：：前不能有空格
