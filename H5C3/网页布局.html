浮动

1、标准流

2、浮动流

浮动格式：选择器 { float：none不浮动，left、right左右浮动 }

浮动的特性：
(1).浮动元素会脱离标准流（脱标）并且不再保留原先的位置，其他盒子会占有这个位置，浮动元素就飘起来了;
(2).浮动会找边沿去浮动，不局限于本行本列;
(3). 浮动元素会一行内显示并且顶部对齐,每一个盒子都要设置浮动才可以，否则就会飘在一个位置;
(4).浮动元素会具有行内块元素特性;
(5).浮动的元素不会压住下面标准流的文字（图片），因为浮动产生的最终目的是为了做文字环绕效果的

为了约束浮动元素的位置，我们一般采用的策略是：
先用标准流的父元素排列上下位置，之后内部子元素采取浮动排列左右位置，符合网页布局的第一准则

清除浮动：消除浮动元素对标准流元素的影响，使浮动元素只在其父级元素内浮动
格式：选择器{ clear：left/right/both},分别为清除左浮动的影响，右浮动，左右两侧浮动

清除浮动方法：

1、额外块级标签法清除浮动:即在浮动元素末尾添加新标签，再将该标签修改样式clear:both
2、给父级元素添加样式：overflow：hidden或auto
3、after伪元素清除法：添加如下代码在css中：
.clearfix:after{
content: "";
display: 0;
clear: both;
visibility: hidden;
}
.clearfix{
*zoom: 1;
}
记得将清除浮动元素的父级元素添加类名clearfix，也可以将这个连同代码一起换名字，实际应用中推荐使用伪元素清除法
4.双伪元素消除法:像3一样添加代码如下：
.clearfix:before, .clearfix:after {
content: "";
display: table;
} .clearfix:after {
clear: both;
} .clearfix {
*zoom: 1;
}

文字围绕浮动元素的巧妙运用：
根据浮动的特性，文字会围绕浮动图片等，只需要给图片设置浮动即可

注意点：

1.多个块级元素纵向排列找标准流，多个块级元素横向排列找浮动流（浮动流块级元素排一起时可以没有间隔等）

2.网页布局第二准则：先设置盒子大小，之后设置盒子位置

3.不建议设计父盒子的高度，应该由子盒子撑开，清除浮动考虑

定位

定位=定位模式+边偏移

定位模式分为四个，通过css的position属性来设置：
1、static静态定位
2、relative相对定位
是元素在移动位置的时候，相对于它原来的位置来说的
格式：选择器 {
position：relative；
top:10px
}
原来的位置仍然保存在原来的标准流中，移动后的位置也以标准流形式对待。
3、absolut绝对定位
是元素在移动位置的时候，相对于它的祖先元素来说的，如果没有父元素或者父元素没有使用定位，则相对于浏览器来定位:如果父元素有使用定位，则以最近一级有定位的祖先元素为参考位置；
绝对定位是脱标的不占有原来的位置
加了绝对定位的盒子可以直接设置宽度和高度，不需要转换为行内块元素或块级元素

口诀：子绝父相，则使用绝对定位的元素，其父元素要用相对定位

注意：加了绝对定位的盒子是不能使用margin：0 auto来设置水平居中的。
此时设置水平居中方法方法：
1、设置left：50%，让盒子的左侧移动到父元素的中心位置，
2、margin-left：-盒子自身宽度的一半

4、fixed固定定位：可以在浏览器滚动时元素的。
如果父亲没有设置定位，爷爷设置了定位，则儿子的绝对定位是现对于爷爷来说的

固定定位是脱标的，是一种特殊的绝对定位。
让固定定位贴合在版心的右（左）侧：
1、让固定定位的盒子left:50%,走到浏览器可视区域的一半。
2、让固定定位的盒子margin-left（right）：版心宽度的一半距离

5、sticky粘性定位：
1、以浏览器的可视窗口为移动参考，
2、粘性定位占有原来的位置，
3、必须添加top等其中一个语句才起作用（即移到什么地方的时候不再滚动）

定位叠放次序 z-index：
1；使用定位时出现盒子重叠情况，用这个来控制盒子叠放次序，
2、数值越大，盒子越靠上，相同则后来者居上，
3、只有设置了定位的盒子才有该属性

定位的特殊性：
1、行内元素添加了绝对定位或者固定定位后，就可以直接设置高度宽度值
2、块级元素添加绝对定位或者固定定位后，如果不给宽度或者高度，则默认大小是内容的大小
3、绝对定位或固定定位会压住下面标准流的全部内容

4、固定定位和绝对定位设置时，与float不能同时起作用，，float的优先级更高

边偏移，就是定位的盒子移到最终位置，有四个值top，button，right，left，四个值分别表示相对于父元素各边的距离，例如：top：88px
